## 为什么 macOS 自带的壁纸不够用

macOS Sonoma 之后，系统自带的锁屏动画确实好看了不少，太阳系、地球这些预设场景会随时间变化光影。问题是，这些场景是苹果写死的，用户没法把自己的视频换上去。

桌面端也是同理。系统设置里能选的"动态壁纸"其实只是会随昼夜切换明暗的静态图，离真正的视频壁纸还差一截。

想要 Windows 上 Wallpaper Engine 那种体验——把任意视频设成动态背景、还能调亮度调音量——在 Mac 上能选的工具其实不多，能做到锁屏联动的，目前就只有 Backdrop。

## Backdrop 是什么？一句话说清楚

**Backdrop 是瑞典开发者 Cindori 推出的 Mac 桌面与锁屏动态壁纸管理工具，支持 4K 视频背景循环播放、多屏独立设置，并且是第一款能将自定义视频同步显示在锁屏界面的 macOS 应用。**

它不是简单的"换图工具"，更像一个迷你的视频壁纸引擎：内置社区库里有上千个用户上传的场景，从治愈系的雨夜到抽象的流体动画都有，也支持导入自己拍的视频直接做成壁纸。

值得一提的是，开发商 Cindori 此前做过 Sensei、Trim Enabler 这类小而精的 Mac 工具，团队规模不大，更新频率反而挺勤。

## Backdrop 核心功能拆解

围着"Mac 动态壁纸"这个需求，Backdrop 主要解决了三类痛点。

**画质与性能的平衡。**很多人担心动态壁纸会让笔记本发热、掉电快。Backdrop 用了自研的视频编码方案，官方给出的数据是 MacBook Pro 上运行时 CPU 占用低于 0.3%，靠 GPU 硬件加速完成解码，这点和早期那些靠 CPU 硬解的山寨壁纸软件完全不是一个量级。

**锁屏视频壁纸（独家功能）。**这是 Backdrop 2.0 版本主推的卖点——选中一个场景，一键应用到桌面和锁屏，两边联动播放，过渡很顺。开发者 Oskar Groth 提到，这套方案是通过对 macOS 壁纸系统进行逆向研究才实现的，目前没有第二款软件做到。

**多屏与社区资源。**接了外接显示器的人可以给每块屏幕单独配置场景，互不干扰；找不到合适素材的话，社区平台里有上千个其他用户分享的成品，搜索时能按风格、色调、分辨率筛选。

简单来说：Backdrop 解决的是"系统自带壁纸太单调，第三方工具又怕费电"这个矛盾，用一套低功耗的视频引擎做到了两边兼顾。

## 和其他 Mac 动态壁纸工具比起来怎么样

围绕"Mac 动态壁纸"这个搜索词，市面上其实还有 Plash（把网页设成壁纸）、Dynamic Wallpaper（视频壁纸库）这类竞品。差异点很明确：

- Plash 免费开源，但本质是网页壁纸，不支持视频，也没有锁屏功能；
- Dynamic Wallpaper 同样主打视频壁纸库，价格亲民，但没有锁屏联动这个独家功能；
- Backdrop 的差异化卡位很清晰——锁屏视频壁纸是目前唯一选项，加上低 CPU 占用的硬件加速引擎。

如果你的需求只是换个好看的桌面图，免费工具够用；如果你想要桌面和锁屏一起动起来，目前能选的就只有 Backdrop 这一个。
